# P3D-002 — Chunks Light : Light, LightDirection, LightPosition, LightShadow, LightGroup

- **Status:** TODO
- **Priority:** P2
- **Module:** P3D
- **Depends on:** —
- **Files:** `dev/codegen/p3d.json:886-903,990-995` (les 5 **déjà décrits**), `src/P3D/P3DChunk.h:62-65,18` (ChunkType), `src/P3D/P3D.generated.{h,cpp}` (à régénérer), `src/Render/`

## Contexte

Les 5 chunks d'éclairage sont **déjà entièrement décrits dans `p3d.json`** :

- `Light` (0x13000) : `version u32`, `name string`, `color vec4`, `intensity float` ;
- `LightDirection` (0x13001) : `direction vec3` ;
- `LightPosition` (0x13002) : `position vec3` ;
- `LightShadow` (0x13004) : `enable u32` ;
- `LightGroup` (0x2380) : `version u32`, `name string`, `numLights u32`,
  `lights: children Light`.

Mais le codegen n'a **pas été relancé** : aucune classe `Light*` dans `P3D.generated.h`.
Nécessaire pour phares de véhicules et lumières dynamiques.

## Approche

1. Vérifier que `Light` agrège bien ses sous-chunks `LightDirection` / `LightPosition` /
   `LightShadow` : actuellement `p3d.json` ne déclare pas ces sous-chunks comme enfants de
   `Light`. Ajouter dans la définition de `Light` :
   `direction: child<vec3> LightDirection`, `position: child<vec3> LightPosition`,
   `shadow: child<u32> LightShadow` (motif `child<T>` déjà utilisé pour `PhysicsVector`,
   `SceneGraphSortOrder`). Reverser le type de lumière (directionnelle / ponctuelle / spot).
2. Lancer `dev/codegen/run.bat`, vérifier les classes `Light`, `LightGroup` et leurs
   `Load()`.
3. Brancher dans `Render/` : exposer les lumières (couleur, intensité, direction/position)
   au pipeline de rendu ; `SceneGraphLightGroup` (0x120109) référence les `LightGroup`.

## Critères d'acceptation

- [ ] Après `run.bat`, `class Light`, `class LightGroup` (+ direction/position/shadow) existent.
- [ ] `Light` expose direction et/ou position quand les sous-chunks sont présents.
- [ ] Un `.p3d` contenant un `LightGroup` se parse sans assert.
- [ ] Les lumières sont récupérables côté `Render/` (couleur + intensité au minimum).
